## Run Locally

Clone the project

```bash
  git clone https://github.com/mohy404/Weather-App-Task.git
```

Go to the project directory

```bash
  cd Weather-App-Task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
